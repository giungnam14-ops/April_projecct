처음 시작을 일단 메모장에 다음처럼 적고 .html로 저장한 후 메모장으로 열고 크롬으로 열어 두 화면을 좌우에 두고 메모장에서 수정하는 대로 나타나는 모습으로 실습해 봅시다.
1. 빈 메모장을 index.html로 저장하고
2. 다른 빈 메모장을 열어 index.html 을 끌어다 놓아 편집기로 사용하고
3. index.html을 크롬화면으로 열어 놓고
4. 두 화면을 화면 분할로 나란히 놓고
5. <h1>나의 첫 번째 웹경험</h1> 을 타이핑하여 저장 후 크롬화면 새로고침
6. <h2>html 부터 시작합니다</h2> 을 타이핑하여 저장 후 크롬화면 새로고침
7. <h3 style="color: red;">css 는 화장하는 기술</h3>
8. <style>
  h3 {
    color: red;                /* 립스틱: 강렬한 포인트 컬러 */
    font-size: 32px;           /* 마스카라/아이라인: 폰트를 크고 뚜렷하게 */
    background-color: #fce4ec; /* 파운데이션: 화사한 핑크빛 톤업 베이스 */
    padding: 15px;             /* 스킨케어: 여백을 주어 촉촉하고 여유롭게 */
    border-radius: 10px;       /* 쉐딩/하이라이터: 모서리를 둥글게, 윤곽 교정 */
    text-align: center;        /* 얼굴 대칭 맞추기: 중앙 정렬로 안정감 부여 */
  }
</style>
